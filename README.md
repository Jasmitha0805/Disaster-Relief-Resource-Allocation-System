 🌍 Disaster Relief Resource Allocation System:
-->A simple web-based application designed to improve coordination during emergencies by managing disaster reports, resource allocation, and relief requests.

📌 Problem Statement:
-->During disasters, relief distribution often lacks coordination, leading to delays, resource wastage, and unmet needs.

🎯 Objective:
*To build a user-friendly system that helps:
	•	Report disasters quickly
	•	Request help efficiently
	•	Manage relief resources
	•	Track and visualize data in real-time
